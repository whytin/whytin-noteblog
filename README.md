# myblog

一个Flask框架搭建的简易博客

### 0、演示

演示地址 https://wakingup.herokuapp.com/


### 1、功能
支持用户登录/文章/说说/标签/评论/友链/api等

<br>

### 2、架构
- 整体：Python2.7.14+Flask+SQLAlchemy
- 前端：Bootstrap+CSS+Flask-WTF
- 文章编辑器：Flask-SimpleMDE
- 后台管理：Flask-Admin

<br>
